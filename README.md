# Dev Template

個人開発で使用する（した）開発環境を構築できるようにしました．

そのままクローンするだけで使えます．

使い方等はまとめておきますので，使えそうであれば使ってみてください．

# 概要

個人開発をしていく上で，開発をするときの環境が __マルチプラットフォーム__ であったり，一度経験したことがある __フレームワーク__ であったりと，割と固定化されていました．

その都度環境をコピペして作ったりするのが煩わしく感じましたので __テンプレート化__ します．

# 前提

この環境を使用するうえで，前提となっている環境があるのでまとめます．

以下の環境がご用意されていると使えます．

- `Git` を使用できる
- `Docker` を使用できる
- `VSCode` を使用できる
- `VSCode` で拡張機能 `Dev Containers` を使用できる

# 使いかた

まず初めに， `README.md` と同階層に `.env` を作成してください．

次に， `Dev Contaiens` 上のコンテナに入って指定のコマンドを実行してください．

以下のコマンドが動いた場合，環境構築は終了です．

お疲れさまでした．

```bash
pip install -r requirements.txt
```
